APLICATIVO WEB PARA LA GESTIÓN DE ATENCIONES DEL CENTRO DE CONTROL *********************

- El aplicativo debera permitir generar solicitudes de atencion
- El aplicativo debera manejar una base de datos de todas las solicitudes, abiertas, atendidas, cerradas y canceladas
- El aplicativo debera permitir descargar un pdf resumen de la atención brindada
- El aplicativo debera mostrar un formulario para crear las solicitudes
- Los inputsbox del aplicativo deberan tener validacion de datos de entrada
- El aplicativo debera permitir actualizar el estado de las atenciones
- El aplicativo debera permitir agregar un comentario al actualizar una atención (obligatorio)
